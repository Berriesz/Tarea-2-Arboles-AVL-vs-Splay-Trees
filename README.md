# Tarea-2-Arboles-AVL-vs-Splay-Trees

## Requisitos

Para ejecutar el proyecto se requiere:

* CLion
* CMake 3.30 o superior
* GCC (MinGW)
* Estándar C11

---

## Configuración en CLion

Al abrir el proyecto por primera vez, es necesario que CLion reconozca el archivo `CMakeLists.txt`.

Si esto no ocurre automáticamente:

1. Abrir el archivo `CMakeLists.txt`.
2. Hacer clic derecho sobre el archivo.
3. Seleccionar **Reload CMake Project**.

   



Una vez cargado correctamente, CLion creará la configuración de ejecución `TAREA2ALGORITMOS`.



## Ejecutar el proyecto

Seleccionar la configuración:
TAREA2ALGORITMOS


y presionar:
Run



## Archivos generados

La ejecución del programa genera tres archivos CSV:
base_results.csv
sequential_results.csv
working_set_results.csv


Estos archivos se guardaran en la carpeta cmake-build-debug (creada automaticamente por clion).
